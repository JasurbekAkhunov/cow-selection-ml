# Прогнозирование удоя и качества молока для отбора коров

## Постановка задачи

Заказчик — **фермер, владелец молочного хозяйства «Вольный луг»**. Необходимо разработать две модели машинного обучения, которые помогут объективно отбирать коров у поставщика «ЭкоФерма»:

- **Модель регрессии** — прогноз годового удоя коровы (целевой признак: `удой, кг`)
- **Модель классификации** — прогноз вкуса молока (целевой признак: `вкус молока`)

Критерии отбора: удой **≥ 6 000 кг/год** и молоко **вкусное**.

## Данные

| Файл | Описание |
|------|----------|
| `ferma_main.csv` | Текущее стадо фермера: характеристики коров, корма, молока |
| `ferma_dad.csv` | Имена пап коров (для дополнительного признака) |
| `cow_buy.csv` | Коровы «ЭкоФермы» — кандидаты на покупку |

### Признаки

| Группа | Признаки |
|---|---|
| Характеристики коровы | Порода, порода папы-быка, возраст |
| Характеристики корма | ЭКЕ, сырой протеин, СПО |
| Характеристика пастбища | Тип пастбища |
| Характеристики молока | Удой (кг), жирность (%), белок (%), вкус молока |

## Этапы работы

### 1. Загрузка и предобработка данных
- Нормализация названий столбцов
- Устранение дубликатов и пропусков
- Приведение числовых признаков к типу `float`
- Кодирование целевого признака классификации

### 2. Исследовательский анализ данных
- Описательная статистика и boxplot-анализ числовых признаков
- Выявление выброса в удое (одна корова с аномально высоким значением)
- Распределение категориальных признаков
- Сравнение средних значений в разрезе вкуса молока

### 3. Корреляционный анализ
- Корреляционная матрица числовых признаков
- Диаграммы рассеяния удоя и признаков корма с разбивкой по породе
- Выявление нелинейной связи СПО → удой (два кластера)
- Выявление нелинейной связи ЭКЕ → удой

### 4. Регрессионные модели (прогноз удоя)

- Feature engineering: бинаризация `СПО`, новый признак `ЭКЕ²`
- Кодирование категорий через `OneHotEncoder`, масштабирование через `StandardScaler`
- Анализ остатков для каждой модели
- Доверительный интервал прогноза лучшей модели

### 5. Классификационная модель (прогноз вкуса молока)
- `LogisticRegression` с настройкой порога классификации
- Метрики: `accuracy`, `recall`, `precision`, матрица ошибок
- Приоритет — минимизация ошибки второго рода (не купить корову с невкусным молоком)
- Настройка порога для обнуления критической ошибки

### 6. Итоговый отбор коров
- Объединение прогнозов удоя и вкуса в один датафрейм
- Фильтрация по критериям заказчика: удой ≥ 6 000 кг и вкусное молоко
- Практические рекомендации фермеру

## Ключевые выводы

- Кормовые показатели (ЭКЕ, сырой протеин) — наиболее значимые предикторы удоя
- Линейная модель с базовым набором признаков оказалась наиболее устойчивой
- Feature engineering (нелинейные преобразования) не улучшил качество на тесте из-за переобучения
- Для классификации вкуса молока важна высокая `precision` — критична ошибка второго рода
